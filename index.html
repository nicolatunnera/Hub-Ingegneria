MTBF = T / n = ore totali / guasti

λ (Failure Rate) = 1 / MTBF = n / T

R(t) = e-λt (affidabilità esponenziale)

MTTR = Tempo medio di riparazione (da inserire separatamente)

Disponibilità A = MTBF / (MTBF + MTTR)

I valori sono indicativi e presuppongono una distribuzione esponenziale dei guasti (modello à tasso costante).
